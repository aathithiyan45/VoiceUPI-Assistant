# Voice UPI Assistant

A smart Android application designed to help users make UPI payments using voice commands.
Built especially for accessibility, useful for visually impaired and elderly users.

---

## Features

* Voice-based payment confirmation
* QR code scanning using CameraX
* ML Kit barcode detection
* Biometric authentication support
* Text-to-Speech feedback
* Simple and user-friendly interface

---

## Tech Stack

* Kotlin
* Android SDK
* CameraX
* Google ML Kit
* Text-to-Speech (TTS)
* SpeechRecognizer API

---

## Setup Instructions

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/VoiceUPI-Assistant.git

2. Open in Android Studio

3. Sync Gradle

4. Run the app on an emulator or real device

---

## Requirements

* Android Studio 
* Android device or emulator
* Internet connection
* Microphone and Camera permissions

---

## Notes

* Enable microphone permission for voice commands
* Camera permission is required for QR scanning
* Works best on a real device
